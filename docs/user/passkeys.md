# Anmelden mit Passkey (ohne Passwort)

Mit einem **Passkey** meldest du dich bei bin-dabei.ch **ohne Passwort** an – per Fingerabdruck, Gesichtserkennung (Face ID / Touch ID / Windows Hello) oder Geräte-PIN. Das ist bequemer (nichts zu merken) und sicherer (nicht durch Phishing abgreifbar).

Passkeys sind **freiwillig** und **zusätzlich**: Dein Passwort funktioniert unverändert weiter. Du kannst dich also je nach Gerät mal per Passkey, mal per Passwort anmelden.

## Einen Passkey einrichten

1. Melde dich wie gewohnt mit deinem **Passwort** an.
2. Öffne oben rechts das **Benutzermenü** (dein Name) › **Passkeys**.
3. Gib einen Namen ein (z. B. «iPhone» oder «Notebook») und klicke auf **Passkey hinzufügen**.
4. Dein Gerät fragt nach Face ID / Touch ID / PIN – bestätige. Der Passkey erscheint danach in der Liste.

Du kannst mehrere Passkeys anlegen (z. B. je einen pro Gerät), sie **umbenennen** oder wieder **entfernen**.

## Mit Passkey anmelden

Auf der Login-Seite erscheint – sofern dein Browser Passkeys unterstützt – die Taste **«Mit Passkey anmelden»**. Ein Klick, Bestätigung per Face ID / Touch ID / PIN, und du bist angemeldet, ohne Benutzernamen oder Passwort einzutippen.

## Gut zu wissen

- **Ein Passkey für alle deine Vereine:** Bist du in mehreren Vereinen aktiv, genügt **ein** Passkey – er funktioniert auf allen deinen Vereins-Adressen.
- **Mehrere Geräte:** Auf Apple-Geräten synchronisiert der iCloud-Schlüsselbund deine Passkeys automatisch (am Mac angelegt, am iPhone nutzbar). Auch andere Passwort-Manager stellen Passkeys geräteübergreifend bereit. Wo das nicht der Fall ist, legst du auf jedem Gerät einfach einen eigenen Passkey an.
- **Passwort nicht vergessen:** Dein Passwort bleibt als Alternative bestehen. Der «Erstlogin»- bzw. Passwort-zurücksetzen-Weg funktioniert wie bisher.
