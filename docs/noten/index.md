# Notenverwaltung

Die elektronische Notenverwaltung von bin-dabei hilft dir, die Noten deines Vereins zu organisieren, den Mitgliedern stimmengenau zuzuteilen und in einem Online-Speicher (z.B. Dropbox) abzulegen.

- **[Stimmenaufteilung](/noten/stimmenaufteilung)** – ein Sammel-PDF automatisch in einzelne, benannte Stimmen-PDFs zerlegen.
